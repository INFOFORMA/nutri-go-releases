# NutriGo — Releases

Ce repo contient uniquement les **artefacts** de l'app NutriGo :
- `version.json` — métadonnées de la dernière version (utilisée pour OTA)
- `data/tasty-recipes.json` — recettes Tasty pré-fetchées chaque semaine
- Releases — APKs de chaque version

Le code source de l'app est dans un repo séparé (privé).
